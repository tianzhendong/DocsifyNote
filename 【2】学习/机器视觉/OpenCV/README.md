## 说明

本部分内容，0-5为学习[OpenCV中文官方文档](http://www.woshicver.com/)，用C++语言编写

由于python在OpenCV开发的便利，后续决定用python学习OpenCV，同时发现了[CodecWang](http://codec.wang/)的博客，对于OpenCV入门非常有帮助，并且写的很有逻辑性，于是决定转载，6-30均为转载的大佬的博客。
