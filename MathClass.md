# Random

```
Random rand = new Random();

//[0,1) 之间的double类型数据
a = rand.nextDouble();
//随机int类型允许范围之内的整形数据
b = rand.nextInt();
//随机生成[0,1)之间的float类型的数据
c  = rand.nextFloat();
//随机boolean
d = rand.nextBoolean();
//随机[0,10)的int
e = rand.nextInt(10)
```
